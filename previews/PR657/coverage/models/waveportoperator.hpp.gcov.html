      75              : //
      76              : // Linear eigenvalue solver for 2D boundary mode computation using a direct linearization
      77              : // of the transverse curl-curl (Equation 1) and normal curl-curl (Equation 2) equations
      78              : // with the Vardapetyan-Demkowicz substitution e_n_tilde = i*kn*E_n, e_t = E_t.
      79              : //
      80              : // Assembles and solves the generalized eigenvalue problem:
      81              : //
      82              : //   [Att  Atn] [et]           [Btt  Btn] [et]
      83              : //   [ 0   Ann] [en] = lambda  [ 0    0 ] [en]
      84              : //
      85              : // where:
      86              : //   Att = mu_cc^{-1}(curl_t u, curl_t v) - omega^2(eps u, v) + BC-t
      87              : //   Atn = -(mu^{-1} grad_t u, v)
      88              : //   Ann = -(mu^{-1} grad u, grad v) + omega^2(eps u, v) + BC-n (H1 stiffness + mass +
      89              : //          impedance boundary)
      90              : //   Btt = (mu^{-1} u, v)
      91              : //   Btn = -Atn^T (negative transpose coupling from Equation 2)
      92              : //
      93              : // The A matrix is upper block-triangular (Ant = 0). The B matrix has Btn coupling from
      94              : // Equation 2 and Bnn = 0. This is a standard GEP with shift-and-invert.
      95              : //
      96              : // The eigenvector contains [e_t; e_n_tilde] where e_n_tilde = i*kn*E_n. Recovery of
      97              : // E_n = e_n_tilde / (i*kn) is performed in the driver, not here.
